1. install node 18 or higher and npm
2. run npm install in the terminal pointing to the folder you extracted
3. run npm run build in the same terminal
4. run npm install -g pm2 in the same terminal
5. run pm2 start npm --name "ollama-ai-ui" -- start in the same terminal
6. enjoy on http://localhost:3000
